# Java Coursework

A collection of Java programs covering fundamental to advanced computer science topics, organized by subject area.

## Repository Structure

| Folder | Topic | Description |
|--------|-------|-------------|
| `01-Basics` | Java Fundamentals | Variables, data types, Scanner input, basic class structure |
| `02-Conditionals-and-Loops` | Control Flow | if-else, switch-case, for/while/do-while loops |
| `03-Arrays-and-Methods` | Arrays & Methods | 1D/2D arrays, static methods, method overloading, linear search |
| `04-OOP-Fundamentals` | Object-Oriented Programming | Classes, objects, constructors, encapsulation, getters/setters |
| `05-Inheritance-and-Polymorphism` | Inheritance & Polymorphism | Class hierarchies, method overriding, polymorphic arrays |
| `06-Abstract-Classes-and-Interfaces` | Abstraction | Abstract classes, interfaces, multiple interface implementation |
| `07-Generics-and-Collections` | Generics & Collections | Generic classes, ArrayList, LinkedList, type-safe collections |
| `08-Exception-Handling` | Exception Handling | try-catch-finally, custom exceptions, input validation |
| `09-File-IO` | File I/O | FileWriter, BufferedReader, reading/writing text files |
| `10-GUI-Swing-AWT` | GUI Programming | AWT & Swing components, event handling, layout managers |
| `11-Concurrency-and-Threads` | Multithreading | Thread lifecycle, synchronization, locks, semaphores, deadlock |
| `12-Data-Structures-HashTable` | Data Structures | Linear Probing & Robin Hood hash table implementations |
| `13-Projects` | Mini Projects | Calculator, number guessing game, matrix multiplication simulator |

## Topics Covered

- **Languages:** Java
- **Paradigms:** OOP, procedural, concurrent programming
- **Data Structures:** Arrays, ArrayList, LinkedList, Hash Tables (Linear Probing, Robin Hood)
- **Concepts:** Generics, Exception Handling, File I/O, GUI (Swing/AWT), Multithreading, Synchronization

## How to Run

Each `.java` file is self-contained with its own `main` method. Compile and run any file:

```bash
javac FileName.java
java FileName
```

> **Note:** Files in `12-Data-Structures-HashTable` work together — compile all three files in that folder.
